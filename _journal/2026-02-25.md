### Gotchas discovered

- **acme.sh standalone mode** — when port 80 has no listener (FrankenPHP only on :443 with `auto_https off`), use `--standalone` instead of `-w /var/www/html`. acme.sh starts its own temp HTTP server for the challenge.
- **FrankenPHP reload not supported** — `Type=simple` doesn't support `systemctl reload`; must use `restart`.
- **mmc git push needs SSH** — HTTPS remote can't push (no device for credential input). SSH remote needs key + known_hosts setup. Simpler to commit locally and push from cachyos.
